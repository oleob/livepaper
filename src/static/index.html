Welcome to livepaper

Currently there are 3 available endpoints:

/api/sky-island

Generates an SVG with the sky island and the current weather. It takes the following parameters:

  • height: height of the svg
  • width: width of the svg
  • City: City from which the weather-data will be retrieved

Example url:

/api/sky-island?height=1000&width=1000&city=Oslo

/api/custom-weather

Generates an SVG with the sky island and a custon weather icon. It takes the followinbg parameters:

  • height: height of the svg
  • width: width of the svg
  • weatherIcon: Code which controles which icon is displayed. For available codes see This link.

Example url:

/api/custom-weather?height=1000&width=1000&weatherIcon=01d

/api/waterbottle

Generates an SVG with a waterbottle which displays a given power-level. It takes the following parameters:

  • height: height of the svg
  • width: width of the svg
  • batteryLevel: A number from 0 to 100

Example url:

/api/waterbottle?height=1000&width=1000&batteryLevel=43
